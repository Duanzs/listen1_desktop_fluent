Listen 1 音乐播放器（Windows无边框版）
=========================

Listen 1可以搜索和播放来自网易云音乐，虾米，QQ音乐三个主流音乐网站的歌曲，让你的曲库更全面。并支持收藏功能，方便的创建自己的歌单。

本版本和原版对比：
- 背景毛玻璃效果（relase也有旧版没有毛玻璃效果的）
- 去除菜单栏
- 去除标题栏及窗口边框
- 所有导航栏空白部分添加拖动控制方便移动窗口
- 所有导航栏空白部分双击最大化，单击出现窗口菜单（最大化、最小化、关闭）
- 修改了一些元素布局，更为美观
